##UKM Theme v6.1.61

Muat turun versi terkini [ukmtheme-master.zip](https://github.com/jrajalu/ukmtheme/archive/master.zip).

###Penerangan
Tema WordPress ini adalah percuma dan boleh digunakan oleh semua Fakulti/Institut/Jabatan/Pusat di Universiti Kebangsaan Malaysia (UKM). Penggunaan secara peribadi atau kumpulan (Persatuan/Kelab) oleh warga UKM amat digalakkan.

###Pemasangan
Pemasangan boleh dilakukan melalui WordPress Dashboard > Appearance > Add New. Klik pautan upload dan pilih [ukmtheme-master.zip](https://github.com/jrajalu/ukmtheme/archive/master.zip) yang telah dimuat turun dan aktifkan. Untuk menetapkan kedudukan kotak-katak di muka hadapan, pergi ke bahagian Theme Options. Tema ini akan memaklumkan sekiranya terdapat kemaskini. Pastikan anda sentiasa mengemaskini kepada versi terkini.
