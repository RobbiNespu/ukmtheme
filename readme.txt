UKM Theme Versi 6.1.9

Diskripsi
=====================================
Tema WordPress ini adalah percuma dan boleh digunakan oleh semua Fakulti/Institut/Jabatan/Pusat di Universiti Kebangsaan Malaysia (UKM). Penggunaan secara peribadi atau kumpulan (Persatuan/Kelab) oleh warga UKM amat digalakkan.

Pemasangan
=====================================
Pemasangan boleh dilakukan melalui WordPress Dashboard atau melalui FTP. Tema ini akan memaklumkan sekiranya terdapat kemaskini. Pastikan anda sentiasa mengemaskini kepada versi terkini.

Kolaborasi
=====================================
Kami mengalu-alukan penglibatan dan sumnagan pengguna untuk memberi pendapat, menguji cuba serta melaporkan ralat untuk penamabahbaikan. Sekiranya terdapat kegagalan atau ralat semasa penggunaan, kami mengharapkan pengguna dapat melaporkan segera ralat tersebut dengan segera kepada pembangun di alamat emel jrajalu@gmail.com
